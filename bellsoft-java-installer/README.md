## Назначение

Проект предназначен для упрощения и стандартизации установки Java на Linux/Unix-системах.

Изначально установка выполнялась с помощью Bash-скрипта `bellsoft_java.sh`, который требовал ручного обновления версии и параметров. В текущей версии проекта основным и рекомендуемым инструментом является Python-скрипт `bellsoft_java.py`, полностью автоматизирующий процесс.

Bash-скрипт сохранён в репозитории **исключительно для истории и сравнения** и не рекомендуется к использованию.

---

## Возможности

- автоматическое получение актуальных версий Java через API BellSoft;
- интерактивный выбор версии, архитектуры и типа пакета;
- поддержка офлайн-режима за счёт локального кэша;
- проверка целостности загружаемых файлов (SHA1);
- автоматическая настройка `JAVA_HOME` и `PATH`;
- создание символической ссылки `latest` на установленную версию.

## Подготовка

Рекомендуется:

- убедиться, что используется Python 3;
- выдать права на выполнение при необходимости:

```bash
chmod +x bellsoft_java.py
```

---

## Использование

### Интерактивная установка

```bash
python3 bellsoft_java.py
```

### Примеры запуска

```bash
# Указание директории установки
python3 bellsoft_java.py --install-dir /opt/java

# Офлайн-режим
python3 bellsoft_java.py --offline

# Отключение цветного вывода
python3 bellsoft_java.py --no-color
```

---

## Конфигурация

Основные параметры доступны через аргументы командной строки:

- `--install-dir` — директория установки Java
- `--work-dir` — директория кэша
- `--offline` — работа без доступа к сети
- `--no-color` — отключение цветного вывода

Подробное описание параметров доступно при запуске с `--help`.

---

## Примечания

- рекомендуется использовать `bellsoft_java.py` как основной инструмент;
- `bellsoft_java.sh` может быть удалён в будущих версиях репозитория;
- установка в системные каталоги может потребовать `sudo`.
